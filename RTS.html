Summary

This research project introduces real-time changes in a custom formation system to enhance formations in real-time strategy games.

It enables users to draw formations in real time, allowing the creation of custom strategies. Users can also remove units from existing formations, automatically adjusting the layout to fit fewer units.

The system uses a gesture recognition algorithm to identify user-drawn formation templates. It improves formations using a grid, filling gaps and correcting minor flaws in the drawing.

It also ensures optimal scaling of the formation based on the number of units, with a margin of 2 units. These real-time adjustments make it possible to integrate the system into existing strategy games for better and more fun formation control.
